Proyecto basado en node.js, typescript, vanillaTs en el frontend, Postgresql.

Es un proyecto para logear los entrenamientos del gimnasio, donde se pueden subir nuevas rutinas, entrenamientos

Quiero los sql para crear la base  (en una carpeta database) y en el mismo repo el frontend y el backend (estan en carpetas distintas).

Para acceder en el desarrollo el frontend tiene que estar en localhost:3000 y el backend en localhost:8000. 

Por ahora no hay seguridad ni usaurios.

Hace una base de datos genérica para después decidir columnas y tablas de esta.
